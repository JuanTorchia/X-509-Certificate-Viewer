## Plugin Name

X.509 Certificate Viewer

## Short Description

A developer tool for inspecting X.509 certificates and Java keystores directly
inside IntelliJ-based IDEs.

## Full Description

X.509 Certificate Viewer helps developers inspect certificates and keystores
without leaving their IDE. Open supported certificate files from the Project
view and review the certificate metadata in a native editor tab.

### Key Features

- Support for certificate files such as `.pem`, `.crt`, `.cer`, and `.der`.
- Support for keystores such as `.p12`, `.pfx`, `.jks`, and `.jceks`.
- Native IntelliJ editor integration.
- Password prompt for encrypted keystores.
- Readable certificate detail view with copy actions for metadata.
- Local inspection workflow; no external upload is required.
